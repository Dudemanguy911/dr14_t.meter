DR14 T.meter is a free and opens source command line tool for computing the Dynamic Range of a given audio file according to the procedure used in the off-line meter released by the Pleasurize Music Foundation.

This tool is very useful for measure how is loud your music and for understanding that a good quality album always has also a good dynamic.

This tool is intended for the Linux users that can not use the Windows application released by the Pleasurize Music Foundation.

How it works:
it simply scan a given directory and it compute the DR value of the audio-files stored in the directory. Actually we support the mp3, flac, m4a and ogg format.
As a result it writes the resulting values in tree files located in the same directory of the audio.files:

dr14.txt
 A tabbed text file
dr14_bbcode.txt
 A table formatted in bbcode (useful for forum)
dr14.html
 An html table, for you site or blog.
