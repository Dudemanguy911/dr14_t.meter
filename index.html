DR14 T.meter is a free and opens source command line tool for computing the Dynamic Range of a given audio file according to the procedure used in the off-line meter released by the Pleasurize Music Foundation.

This tool is very useful for measure how is loud your music and for understanding that a good quality album always has also a good dynamic, and it's also useful for understanding the effects of the so called loudness war.

Dr14 t.meter is released according to the GPL3 and  is intended for the Linux users that can not use the Windows application released by the Pleasurize Music Foundation or the plugin for Foobar200.

I've largely tested this application and it has always returned results identical to those of the official off-line meter.

How it works:
it simply scans a given directory and it compute the DR value of the audio-files stored in the directory. Actually we support the mp3, flac, m4a and ogg format.
As a result it writes the resulting values in three files located in the same directory of the audio.files:

dr14.txt
 A tabbed text file
dr14_bbcode.txt
 A table formatted in bbcode (useful for forum)
dr14.html
 An html table, for you site or blog.

System requirement:
The tool is written in python3 so you must install the python3 package in your system and the libraries numpy and scipy (for python3) as dependencies.
For decoding the audiophiles the packages: lame flac vorbis-tools faad2  are required!

How to use:
Enter in the directory where you have putted the tool and give the command:

./dr14_tmeter [your music directory]

When the computation is finished you will find in your music directory the three files described above. 
I recommend to add the option -m to enable the multi-core mode.

For more information type the command:
./dr14_tmeter --help

Installation:

Enter in the package directory and with the root privileges type the command:
> python3 setup.py install

This command will install the tool in your /usr/bin directory.
After the installation you should use the command dr14_tmeter form the shell.
