Features list:

  • Compute the DR dynamic range of the audio files
  • Multithreaded computation
  • Read mp3 files via lame
  • Read ogg files via vorbis-tools
  • Read mp4 files via ffmpeg
  • Read ape files via ffmpeg
  • Read flac files via flac tool
  • Scan a full directory
  • Scan recursively the subdirectories
  • Write the result in a tabbed text file
  • Write the result in a formatted HTML table
  • Write the result in a formatted BBCODE table
  • Write the resulting tables in the audio file's directory
  • Uses the audio metadata for generating the tables
